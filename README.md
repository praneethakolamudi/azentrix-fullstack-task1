## 🛠️ Tech Stack
- React JS (Vite)
- CSS Animations (IntersectionObserver)
- Vanilla CSS (inline styles)

## ✨ Features
- 5 Sections: Hero, About, Skills, Projects, Contact
- Smooth scroll navigation
- Scroll-triggered animations
- Animated skill progress bars
- Project cards with hover effects
- Contact form with client-side validation
- Fully responsive layout

## ⚙️ Setup Instructions

1. Clone the Repository
git clone https://github.com/praneethakolamudi/azentrix-fullstack-task1.git

2. Go to Project Folder
cd azentrix-fullstack-task1

3. Install Dependencies
npm install

4. Run the Project
npm run dev

5. Open in Browser
http://localhost:5173
